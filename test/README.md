# Test Fixtures

Файлы для ручного тестирования бота. Отправляй в чат и наблюдай поведение.

## Генерация xlsx файлов

```bash
cd personal-finance-assistant
pip install pandas openpyxl
python test/generate_xlsx.py
```

### 💡 Интересные кейсы для проверки после загрузки

После загрузки `01_normal_en.csv` попробуй в боте:
- «Установи лимит 500₽ в неделю на кофе» → должен сработать set_limit
- «Установи лимит 500 рублей в неделю на кофе» → то же самое
- «не хочу тратить больше 1000 в месяц на такси» → LLM должен распарсить
- «Где я трачу больше всего?» → ReAct вызовет get_spending_report
- «Найди дешевле в категории кофе» → find_cheaper
